Restaurant Automation Api Documentation

Root URL : http://restaurantmanagementapi-env.eba-eujhhwnw.ap-south-1.elasticbeanstalk.com/api

API ENDPOINTS

Create New User
endpoint : /user
method : POST
permissions : [ 'owner' ]
body: name ( string , required ) , password ( string , required ) , role ( array , required ) , salary (number , optional)
Get All Users
endpoint : /user
method : GET
permissions : [ 'owner' ]
Update User
endpoint : /user
method : PUT
permissions : [ 'owner' ]
body: id ( string , required) name ( string , optional ) , password ( string , optional ) , role ( array , optional ) , salary (number , optional)
User Login
endpoint : /auth/login
method : POST
permissions : ALL
body: name ( string , required ), password ( string , required )
Create New Inventory
endpoint : /inventory
method : POST
permissions : [ 'inventoryManager' ]
body: inventoryCategory ( string , required ) , itemName ( string , required ) , quatity (number , required ) , measurementUnit (string,array , required), unitRate (number,required)
Get All Inventory
endpoint : /inventory
method : GET
permissions : [ 'inventoryManager' ]
Get Inventory by Category
endpoint : /inventory/:inventoryCategory
method : GET
permissions : [ 'inventoryManager' ]
